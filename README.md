# The Sundering Spire

A two-player asymmetric dungeon crawler / puzzle combat prototype.

Playable GitHub Pages build:

https://turbodog111.github.io/multiplayer-dungeon/

## Current Prototype

The first Pages build is a same-keyboard local test harness. It proves the browser renderer, Sela's light readability, Kade's role, a mirror puzzle, and a mocked co-op takedown chain before the online WebSocket server exists.

## Prototype Controls

Sela:

- Move: `WASD`
- Light / quick attack: `Space` or `F`
- Plant or recall beacon: `Q`

Kade:

- Move: arrow keys
- Strike / rotate nearby prism: `Enter`
- Phase into Echo layer: right `Shift` or `/`

Prototype objective: charge the mirror relay, break the Stone Sentry with the co-op takedown sequence, open the stair gate, and bring both heroes to the stairs.
